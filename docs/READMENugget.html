Butt Client

The framework-agnostic, user-friendly library for your adult toys integration needs!

Usage

To set all your machines to half intensity:

  1. Launch Intiface ©️ Central and connect your toys.
  2. Launch ButtServer.
  3. Add this NuGet package to your project.
  4. Somewhere in your code...
ButtClient client = new ButtClient("ClientName");
client.Set(0.5f);
client.Disconnect();

This a transparent wrapper around the BasicButtManager library, via the ButtServer proxy. The public-facing interface is identical to that of BasicButtManager besides the class name.
